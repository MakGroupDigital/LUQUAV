# Instructions pour ajouter le logo LUQUAV

Pour que le logo s'affiche correctement partout (favicon, partages sociaux, résultats Google), vous devez placer les fichiers suivants dans le dossier `/public` :

## Fichiers nécessaires :

1. **logo.png** (1200x630px) - Pour les partages sociaux (Open Graph, Twitter Cards)
2. **favicon-16x16.png** (16x16px) - Favicon standard
3. **favicon-32x32.png** (32x32px) - Favicon standard
4. **favicon-96x96.png** (96x96px) - Favicon standard
5. **apple-touch-icon.png** (180x180px) - Pour iOS
6. **android-chrome-192x192.png** (192x192px) - Pour Android
7. **android-chrome-512x512.png** (512x512px) - Pour Android
8. **mstile-144x144.png** (144x144px) - Pour Windows
9. **safari-pinned-tab.svg** - Pour Safari (optionnel)

## Format recommandé :

Tous les fichiers doivent être au format PNG (sauf le SVG pour Safari) avec fond transparent ou fond noir selon votre logo.

Le logo principal (logo.png) doit être de 1200x630px pour un affichage optimal sur les réseaux sociaux.

## Emplacement :

Placez tous ces fichiers dans le dossier `/public` à la racine du projet.

Une fois les fichiers ajoutés, le logo s'affichera automatiquement :
- Dans l'onglet du navigateur (favicon)
- Dans les résultats de recherche Google
- Lors des partages sur Facebook, Twitter, LinkedIn, etc.
- Sur les appareils mobiles (iOS, Android)
